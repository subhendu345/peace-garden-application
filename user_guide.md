# Peace Garden App ☁️🌿
A secure cloud deployment using Terraform, AWS, and monitoring tools.

## How to Use
1. Clone the repo
2. Run `terraform init`
3. Edit `terraform.tfvars`
4. Deploy using `terraform apply`

## Modules
- EC2
- VPN with OpenVPN
- Honeypot
- DynamoDB
- Monitoring with CloudWatch
